# gem-me-bot
A slack bot, looking for #gems

## Install

`npm install`

### Setup

Create a `.env` file with your slack token.

    SLACK_TOKEN=YOUR_SLACK_TOKEN
    
Change your "gem" channel to the name of your `gems`. (e.g. `#gems`). 
    
## Run

`npm run start` or `babel-node lib/bot.js`

### Requirements

  - node 4+
  - npm 2+
  - babel
